# Agent Instructions for pomodoro-timer

## Before starting any task
1. Read `CLAUDE.md` for project rules and conventions.
2. Read `LEARNINGS.md` for accumulated project knowledge.
3. Read `openspec/project.md` for architecture and tech stack.
4. Check `openspec/specs/` for existing feature specifications.

## After completing any task
1. If you discovered something new, append it to `LEARNINGS.md`.
2. If you created a new feature, create a spec in `openspec/specs/`.
